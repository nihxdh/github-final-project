# Simple Interest Calculator

This project is a simple calculator that computes **Simple Interest** using a Bash script.

## Purpose
The calculator helps users quickly compute the simple interest based on:
- Principal amount
- Rate of interest
- Time period

## Formula

Simple Interest is calculated using the formula:

SI = (P × R × T) / 100

Where:
- **P** = Principal amount
- **R** = Rate of interest per year
- **T** = Time period in years
